## Checklist Before Submitting

- [ ] Plugin works with the latest stable Notesaner version
- [ ] Manifest is complete and valid
- [ ] All declared capabilities are actually used
- [ ] Plugin has a README with screenshots
- [ ] Plugin has a LICENSE file
- [ ] If `network` capability is used: privacy policy URL is in manifest

## Submission Process

1. Fork [notesaner/plugin-registry](https://github.com/notesaner/plugin-registry)
2. Add your plugin entry to `plugins.json`
3. Open a pull request with the PR template
4. Wait for review (2–7 business days)
5. Once approved, your plugin appears in the registry
