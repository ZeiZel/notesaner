# Contributing Overview

Thank you for your interest in contributing to Notesaner! This guide covers all the ways you can get involved.

## Ways to Contribute

### Code Contributions

- Fix bugs reported in [GitHub Issues](https://github.com/notesaner/notesaner/issues)
- Implement features from the [roadmap](https://github.com/notesaner/notesaner/discussions)
- Improve performance, accessibility, or security

### Documentation

- Fix typos and improve clarity
- Write missing pages (pages marked "Coming Soon")
- Add examples and screenshots
- Translate documentation

### Plugins

Build and publish plugins to the community registry. See [Plugin Development Guide](/docs/plugin-development/getting-started/architecture).

### Testing

- Report bugs with detailed reproduction steps
- Run E2E tests on your platform
- Test release candidates before they ship

## Getting Started

1. Read [Development Setup](/docs/contributing/dev-setup)
2. Browse [good first issues](https://github.com/notesaner/notesaner/issues?q=label%3A%22good+first+issue%22)
3. Read [Submitting a Pull Request](/docs/contributing/pull-request)

## Code of Conduct

Please read our [Code of Conduct](/docs/contributing/code-of-conduct) before participating.
