# Issue Reporting

## Before Reporting

1. Search [existing issues](https://github.com/notesaner/notesaner/issues) to avoid duplicates
2. Try to reproduce with the latest version
3. Check the [troubleshooting guides](/help/troubleshooting/contact-support)

## Bug Reports

Include:

- **Notesaner version** (Settings → About)
- **Browser and OS** (e.g. Chrome 123 on Ubuntu 24.04)
- **Steps to reproduce** (numbered list)
- **Expected behavior**
- **Actual behavior**
- **Screenshots or screen recording** (highly recommended)
- **Console errors** (F12 → Console, copy any red errors)

## Feature Requests

Include:

- **Problem you're trying to solve** (not just the solution)
- **Proposed solution** (optional)
- **Alternatives you've considered**
- **Any prior art** (Obsidian, Notion, Logseq equivalents)

## Security Vulnerabilities

Do NOT open a public GitHub issue for security vulnerabilities. Email security@notesaner.com instead.
