# Calendar View

:::info Coming Soon

The Calendar plugin adds a monthly calendar view to the sidebar. Days with existing notes are highlighted. Click any day to open or create a daily note for that date.

See [Calendar & Daily Notes Plugin](/help/plugins/calendar-daily-notes) for setup instructions.
