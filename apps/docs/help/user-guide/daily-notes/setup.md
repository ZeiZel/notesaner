# Daily Notes Setup

:::info Coming Soon
This page is under construction. Complete documentation will be available soon.

Daily Notes automatically creates a note for today's date when you open the app (or on demand).

## Enabling

Go to **Settings → Plugins → Daily Notes** and toggle it on.

## Configuration

| Setting             | Description                                   |
| ------------------- | --------------------------------------------- |
| **Date format**     | Filename format, e.g. `YYYY-MM-DD`            |
| **Folder**          | Where daily notes are stored, e.g. `Journal/` |
| **Template**        | Note template used for new daily notes        |
| **Open on startup** | Auto-open today's note on app launch          |
