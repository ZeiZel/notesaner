## Filter Panel

Open the filter panel with `Cmd+Shift+F` / `Ctrl+Shift+F` to build metadata filters visually.

## Quick Tag Filter

Click any tag in the Tags panel to instantly filter the note list.

## Advanced Filters

Use [Advanced Search Syntax](/help/user-guide/search/advanced-syntax) for complex property-based queries:

```
property:status = "published" AND tag:blog after:2026-01-01
```
