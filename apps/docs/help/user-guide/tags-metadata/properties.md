# Properties (Front Matter)

:::info Coming Soon
This page is under construction. Complete documentation will be available soon.
:::

Properties are key-value pairs stored in YAML front matter at the top of a note. See [Front Matter & Properties](/help/user-guide/editor/front-matter) for the full reference.

## Property Types

| Type    | Example                 |
| ------- | ----------------------- |
| Text    | `status: "in-progress"` |
| Number  | `priority: 1`           |
| Date    | `due: 2026-04-01`       |
| Boolean | `published: true`       |
| List    | `tags: [a, b, c]`       |
