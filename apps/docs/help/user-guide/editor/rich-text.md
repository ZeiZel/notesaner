The Notesaner editor is powered by TipTap, providing a rich WYSIWYG editing experience on top of Markdown.

## Using the Bubble Menu

When you select text, a floating bubble menu appears with formatting options:

- Bold, Italic, Strikethrough
- Inline code
- Link insertion
- Heading level

## Using the Slash Menu

Type `/` to open the block insertion menu. You can insert:

- Headings (H1–H6)
- Bullet and numbered lists
- Code blocks
- Tables
- Callouts
- Images
- Dividers
