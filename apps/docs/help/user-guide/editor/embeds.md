## Images

Drag an image file into the editor to upload and embed it. You can also use the slash menu (`/image`) or Markdown syntax:

```markdown
![Description](./attachments/screenshot.png)
```

## PDFs

Drag a PDF into the editor to create an inline PDF viewer. Supported display modes:

- Inline embed (full document preview)
- Link with thumbnail

## Videos

Video embeds support local video files and external URLs (YouTube, Vimeo):

```markdown
![](https://www.youtube.com/watch?v=dQw4w9WgXcQ)
```

## Attachments Folder

Uploaded files are stored in an `attachments/` subfolder relative to the note file.
