## Creating a Table

Type `/table` in the editor and select **Table** from the slash menu, or type Markdown table syntax directly.

## Editing Tables

- **Add row**: Click the `+` button below the last row, or press `Tab` in the last cell
- **Add column**: Click the `+` button to the right of the last column
- **Delete row/column**: Right-click a cell for the context menu
- **Merge cells**: Select multiple cells and right-click → Merge

## Sorting

Click a column header to sort the table by that column. Click again to reverse the sort order.
