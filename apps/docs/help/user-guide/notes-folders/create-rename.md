## Creating Notes

- **Keyboard shortcut**: `Cmd+N` / `Ctrl+N`
- **Sidebar button**: Click the `+` icon in the sidebar
- **Right-click**: Right-click a folder → **New Note**

## Renaming Notes

Double-click the note title in the editor header, or right-click the note in the sidebar → **Rename**.

Note filenames must be unique within a folder. Renaming a note automatically updates all `[[wikilinks]]` that reference it.

## Deleting Notes

Right-click a note in the sidebar → **Delete**. Deleted notes are moved to the **Trash** and can be restored within 30 days.
