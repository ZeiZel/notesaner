## Uploading Files

Drag any file into a note to upload and attach it. Files are stored in the `attachments/` subfolder relative to the note.

## Attachment Manager

Open the **Attachment Manager** from the sidebar to see all files across your workspace, find orphaned attachments, and delete unused files.

## Supported File Types

- **Images**: PNG, JPG, GIF, WEBP, SVG
- **Documents**: PDF
- **Video**: MP4, WebM
- **Audio**: MP3, OGG, WAV
