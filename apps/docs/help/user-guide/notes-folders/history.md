## Version History

Every change to a note is automatically versioned. Open the **History** panel via the note menu (⋯) → **View History**.

The history panel shows:

- Timestamp of each version
- Author who made the change
- A diff view showing what changed

## Restoring a Version

Click any version in the history panel and select **Restore** to revert the note to that state. The current version is preserved as a new history entry.

## Trash

Deleted notes are moved to the Trash. To restore a deleted note:

1. Open **Workspace Settings → Trash**
2. Find the note
3. Click **Restore**

Notes in the Trash are permanently deleted after 30 days.
