## Drag and Drop

Drag notes and folders in the sidebar to reorganize them. Folders can be nested up to any depth.

## Move Command

Use `Cmd+Shift+M` / `Ctrl+Shift+M` to move the current note to a different folder via a fuzzy-search picker.

## Bulk Select

Hold `Cmd` / `Ctrl` and click multiple notes in the sidebar to select them, then drag or right-click → **Move to...**.
