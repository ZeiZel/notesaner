# Backlinks

Backlinks show every note that contains a link to the current note. They help you discover context and connections you may have forgotten.

## Backlinks Panel

The **Backlinks** panel is in the right sidebar. It shows:

- The linking note's title
- The exact sentence or paragraph containing the link
- A "ghost" entry for unlinked mentions (text that matches the current note title but isn't a wikilink)

## Unlinked Mentions

Unlinked mentions are occurrences of the current note's title that haven't been turned into explicit links. You can click **Link** next to any unlinked mention to convert it to a wikilink.

## Backlinks Plugin

The Backlinks Panel can be extended via the [Backlinks Plugin](/help/plugins/backlinks-panel).
