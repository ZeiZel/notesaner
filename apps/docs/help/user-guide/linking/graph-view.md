The Graph View shows all your notes as nodes and their wikilinks as edges, creating a visual map of your knowledge base.

## Opening

Press `Cmd+G` / `Ctrl+G` or click the graph icon in the sidebar.

## Controls

- **Pan**: Click and drag the background
- **Zoom**: Scroll wheel or pinch gesture
- **Open note**: Click a node
- **Focus**: Double-click a node to center the graph on it

## Filters

Use the filter panel to restrict what's shown:

- **Depth**: Show only notes N links away from the focused note
- **Tags**: Filter by tag
- **Folder**: Show only notes from specific folders

## Local Graph

Every note has a **Local Graph** view (smaller graph showing only directly connected notes) accessible from the note toolbar.
