# Outgoing Links Panel

:::info Coming Soon
This page is under construction. Complete documentation will be available soon.

The **Outgoing Links** panel shows all wikilinks in the current note, along with a preview of the linked note. It mirrors the Backlinks panel but in the opposite direction.

Open it from the right sidebar → **Outgoing Links**.
