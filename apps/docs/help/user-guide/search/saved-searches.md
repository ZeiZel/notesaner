Save a search query to the sidebar for one-click access. Saved searches act like dynamic folders — they always show notes that match the query at that moment.

## Creating a Saved Search

1. Run a search using the command palette
2. Click **Save Search** in the search results header
3. Give it a name

## Managing Saved Searches

Saved searches appear in the left sidebar under **Saved Searches**. Right-click to rename or delete them.
