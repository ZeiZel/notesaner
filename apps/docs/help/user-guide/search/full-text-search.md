# Full-Text Search

Press `Cmd+K` / `Ctrl+K` to open the command palette / search. Start typing to search note titles and content.

## Search Scope

By default, search covers:

- Note titles
- Note content (full text)
- Tags
- Properties (front matter values)

## Search Results

Results are ranked by relevance. Matching terms are highlighted in the preview. Click a result to open the note and jump to the first match.

## Keyboard Navigation

- `↑` / `↓` — navigate results
- `Enter` — open selected note
- `Escape` — close search
