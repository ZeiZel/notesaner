## Left Sidebar

The left sidebar contains:

- File explorer (folder tree)
- Tags panel
- Search
- Saved searches
- Plugin-added panels

Toggle visibility with `Cmd+\` / `Ctrl+\`.

## Right Sidebar

The right sidebar contains:

- Properties panel
- Backlinks panel
- Outgoing links panel
- Table of contents
- Plugin-added panels

Toggle visibility with `Cmd+Shift+\` / `Ctrl+Shift+\`.

## Reordering Panels

Drag panel headers to reorder them within a sidebar.
