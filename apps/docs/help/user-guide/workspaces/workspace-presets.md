# Workspace Presets

:::info Coming Soon
This page is under construction. Complete documentation will be available soon.
:::

Workspace presets save your current panel layout, open tabs, and sidebar state so you can switch between different working contexts.

## Saving a Preset

**View → Save Workspace Preset** → give it a name.

## Loading a Preset

**View → Load Workspace Preset** → select from the list.

## Use Cases

- **Writing mode**: Distraction-free single panel, no sidebars
- **Research mode**: Graph view + notes side by side
- **Review mode**: All sidebars open, backlinks visible
