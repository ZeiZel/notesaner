## Tabs

Open multiple notes in tabs, similar to a code editor. Click the `+` button in the tab bar to open a new tab.

## Panels

A panel contains one or more tabs. You can have multiple panels open side by side (split views).

## Pinning Tabs

Right-click a tab → **Pin Tab** to keep it visible when switching context.
