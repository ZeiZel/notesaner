Notesaner supports community themes that change colors, typography, and layout.

## Installing a Theme

1. Go to **Settings → Appearance → Themes**
2. Browse the theme gallery
3. Click **Install** on any theme

## Creating a Theme

Themes are CSS files. See [Custom CSS](/help/user-guide/appearance/custom-css) for the base variable reference.
