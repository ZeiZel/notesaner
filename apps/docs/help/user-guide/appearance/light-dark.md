# Light / Dark Mode

Notesaner supports light mode, dark mode, and automatic (follows your OS setting).

## Switching Modes

- **Toolbar button**: Click the sun/moon icon in the top-right corner
- **Settings**: **Settings → Appearance → Color Mode**
- **Keyboard**: `Cmd+Shift+D` / `Ctrl+Shift+D`

## Auto Mode

Set to **Auto** to automatically match your operating system's color scheme preference.
