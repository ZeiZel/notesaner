The Spaced Repetition plugin turns notes into flashcards and schedules reviews using the SM-2 algorithm.

## Creating Flashcards

Add a flashcard to a note using the `:::flashcard` block:

```markdown
:::flashcard
**Q**: What is a CRDT?
**A**: Conflict-free Replicated Data Type — a data structure that enables automatic conflict resolution in distributed systems.
:::
```

## Reviewing

Click **Review Due Cards** in the sidebar to start a review session. After each card, rate your recall (Again, Hard, Good, Easy) to update the next review schedule.
