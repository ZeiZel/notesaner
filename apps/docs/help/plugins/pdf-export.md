## Exporting a Single Note

Go to note menu (⋯) → **Export as PDF**.

## Exporting a Folder

Right-click a folder → **Export as PDF** to combine all notes in the folder into a single PDF.

## Options

| Option                        | Description                        |
| ----------------------------- | ---------------------------------- |
| **Page size**                 | A4, Letter, etc.                   |
| **Include table of contents** | Auto-generated from headings       |
| **Page numbers**              | Toggle footer with page numbers    |
| **Include front matter**      | Whether to include YAML properties |
