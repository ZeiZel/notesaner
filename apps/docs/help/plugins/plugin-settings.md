# Plugin Settings Reference

:::info Coming Soon
This page is under construction. Complete documentation will be available soon.
:::

All plugin settings are accessible at **Settings → Plugins → [Plugin Name]**.

| Plugin                 | Key Settings                       |
| ---------------------- | ---------------------------------- |
| AI Assistant           | Provider, Model, API Key           |
| Calendar & Daily Notes | Folder, Date format, Template      |
| Database View          | Default view, Column mappings      |
| Focus Mode             | Typewriter scrolling, Dim inactive |
| Graph View             | Node size, Link color, Physics     |
| Kanban Board           | Status property, Column order      |
| Slides                 | Theme, Transition, Highlight theme |
| Spaced Repetition      | Algorithm, Daily review limit      |
| Templates              | Templates folder                   |
| Web Clipper            | Default folder, Default tags       |
