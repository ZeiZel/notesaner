# Managing & Updating Plugins

## Plugin Settings Page

Go to **Settings → Plugins** to see all installed plugins.

## Enabling / Disabling

Toggle the switch next to any plugin to enable or disable it. Core plugins can be disabled but not uninstalled.

## Updating

Plugins show an **Update** badge when a new version is available. Click **Update** to install the latest version.

Enable **Auto-update plugins** in Settings to update automatically.

## Uninstalling

Click the **⋯** menu next to a community plugin → **Uninstall**. This removes the plugin files from your workspace.

## Plugin Data

When you uninstall a plugin, its data (settings, stored files) is not automatically deleted. Go to **Settings → Plugins → [Plugin Name] → Clear Data** to remove it.
