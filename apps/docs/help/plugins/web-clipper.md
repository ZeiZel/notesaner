The Web Clipper browser extension (Chrome, Firefox, Safari) lets you save web pages, articles, and highlights directly to Notesaner.

## Installing

1. Go to **Settings → Plugins → Web Clipper**
2. Click the browser extension link for your browser
3. Install the extension
4. Sign in with your Notesaner account

## Clipping

Click the Web Clipper icon in your browser toolbar while on any page:

- **Full page**: Save the entire page as Markdown
- **Article**: Extract the main article content (reader mode)
- **Highlight**: Save selected text with a link to the source
- **Screenshot**: Save a screenshot of the visible area
