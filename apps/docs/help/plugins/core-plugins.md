# Core Plugins Overview

Notesaner ships with 15 built-in plugins. All are enabled by default and can be disabled individually in **Settings → Plugins**.

| Plugin                     | Description                                            |
| -------------------------- | ------------------------------------------------------ |
| **AI Assistant**           | AI-powered writing suggestions, summarization, and Q&A |
| **Backlinks Panel**        | Shows all notes linking to the current note            |
| **Calendar & Daily Notes** | Monthly calendar widget + daily note automation        |
| **Database View**          | Table, Board, and Gallery views on note folders        |
| **Diagrams (Mermaid)**     | Renders Mermaid code blocks as diagrams                |
| **Drawing (Excalidraw)**   | Embedded freehand drawing canvas                       |
| **Focus Mode**             | Distraction-free writing with typewriter scrolling     |
| **Graph View**             | Interactive knowledge graph visualization              |
| **Kanban Board**           | Card-based task tracking on tagged notes               |
| **PDF Export**             | Export notes or folders to PDF                         |
| **Slides / Presentations** | Reveal.js-based presentations from Markdown            |
| **Spaced Repetition**      | Flashcard creation and review scheduling               |
| **Templates**              | Note templates with variable substitution              |
| **Web Clipper**            | Browser extension to clip web pages to notes           |
| **Outgoing Links**         | Shows all links in the current note                    |

## Managing Plugins

See [Managing & Updating Plugins](/help/plugins/manage-plugins) for how to enable, disable, configure, and update plugins.
