This page is under construction. Complete documentation will be available soon.

The Drawing plugin embeds an Excalidraw canvas inside a note.

## Creating a Drawing

Type `/drawing` in the editor and select **Drawing** to insert an embedded canvas.

## Drawing Tools

- Pencil, shapes (rectangle, circle, arrow, line, diamond)
- Text labels
- Eraser
- Selection and move

## Exporting

Right-click the drawing → **Export as PNG** or **Export as SVG**.

## Sharing

Drawings are stored as JSON in a linked `.excalidraw` file next to the note.
