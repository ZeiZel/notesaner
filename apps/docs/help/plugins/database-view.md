# Database View

:::info Coming Soon
This page is under construction. Complete documentation will be available soon.
:::

The Database View plugin treats a folder of notes as a database, displaying them in table, board (kanban), or gallery views.

## Views

| View        | Best for                                                       |
| ----------- | -------------------------------------------------------------- |
| **Table**   | Seeing all notes with their properties in rows and columns     |
| **Board**   | Grouping notes by a property (e.g. `status`) as kanban columns |
| **Gallery** | Browsing notes visually with cover images                      |

## Requirements

Notes in the folder should have front matter properties for the view to display useful columns.
