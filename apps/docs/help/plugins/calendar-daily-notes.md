The Calendar plugin adds two features:

1. **Calendar widget**: A monthly calendar in the sidebar. Days with notes are highlighted. Click a day to open or create a note.
2. **Daily Notes**: Automatically creates a new note for today when you open the app.

## Setup

1. Go to **Settings → Plugins → Calendar & Daily Notes**
2. Set a **daily notes folder** (e.g. `Journal/`)
3. Set a **date format** for filenames (e.g. `YYYY-MM-DD`)
4. Optionally select a **template** for new daily notes
