# Focus Mode

Focus Mode hides all sidebars and UI chrome, leaving only the editor visible.

## Enabling Focus Mode

- **Keyboard**: `Cmd+Shift+F` / `Ctrl+Shift+F`
- **Toolbar**: Click the focus icon in the note toolbar

## Typewriter Scrolling

When enabled, the cursor stays in the middle of the screen as you type. Enable it in **Settings → Plugins → Focus Mode → Typewriter scrolling**.

## Dim Inactive Paragraphs

Optionally dim all paragraphs except the one currently being edited. Enable in **Settings → Plugins → Focus Mode → Dim inactive paragraphs**.

## Exiting Focus Mode

Press `Escape` or the same keyboard shortcut to exit.
