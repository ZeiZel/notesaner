# Export & Data Portability

Notesaner stores your notes as Markdown files. You can export them at any time.

## Export Options

| Format                   | Includes                                             |
| ------------------------ | ---------------------------------------------------- |
| **Markdown (ZIP)**       | All `.md` files with folder structure preserved      |
| **Obsidian Vault (ZIP)** | Same as Markdown — compatible with Obsidian directly |
| **JSON**                 | All notes with metadata as structured JSON           |
| **PDF**                  | All notes rendered to PDF (workspace-level export)   |

## Requesting an Export

1. Go to **Settings → Account → Export Data**
2. Select your preferred format
3. Click **Request Export**
4. You'll receive an email with a download link within a few minutes

## Data Portability

Your data is always portable. Notes are stored as `.md` files that can be opened in any text editor or Markdown application.
