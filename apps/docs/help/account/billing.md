## Plans

| Plan           | Storage   | Collaborators | Features                            |
| -------------- | --------- | ------------- | ----------------------------------- |
| **Free**       | 1 GB      | 3             | Core features                       |
| **Pro**        | 20 GB     | 10            | All plugins, version history        |
| **Team**       | 100 GB    | 50            | SSO, admin tools, audit log         |
| **Enterprise** | Unlimited | Unlimited     | Self-hosting, SLA, custom contracts |

## Self-Hosted

Self-hosted instances are free and have no user or storage limits. See [Self-Hosting Guide](/docs/self-hosting/overview).
