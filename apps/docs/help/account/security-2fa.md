## Enabling 2FA

1. Go to **Settings → Account → Security**
2. Click **Enable Two-Factor Authentication**
3. Scan the QR code with an authenticator app (Google Authenticator, 1Password, Authy)
4. Enter the 6-digit code to verify and save

## Recovery Codes

After enabling 2FA, download your recovery codes and store them safely. Recovery codes can be used to log in if you lose access to your authenticator app.

## Active Sessions

View and revoke active login sessions from **Settings → Account → Security → Active Sessions**.
