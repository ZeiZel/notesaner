## Plugin Failed to Load

If a plugin shows an error:

1. Try disabling and re-enabling the plugin
2. Check if there's an update available (**Settings → Plugins → Check for Updates**)
3. Try reinstalling the plugin

## Reporting a Plugin Bug

For community plugins, open an issue on the plugin's GitHub repository.
For built-in plugins, see [Contact Support](/help/troubleshooting/contact-support).
