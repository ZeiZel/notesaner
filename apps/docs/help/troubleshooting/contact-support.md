# Contact Support & Report a Bug

## Bug Reports

To report a bug, open a GitHub issue:

1. Go to [github.com/notesaner/notesaner/issues](https://github.com/notesaner/notesaner/issues)
2. Click **New Issue**
3. Select **Bug Report**
4. Fill in the template:
   - What you expected to happen
   - What actually happened
   - Steps to reproduce
   - Browser, OS, and Notesaner version

## Feature Requests

Use the same GitHub issue tracker with the **Feature Request** template.

## Community Discussion

Join the [GitHub Discussions](https://github.com/notesaner/notesaner/discussions) for questions, tips, and community support.

## Self-Hosted Instance Issues

For self-hosting issues, see the [Self-Hosting Troubleshooting Guide](/docs/self-hosting/troubleshooting).
