# Keyboard Shortcuts

## Global

| Action          | macOS   | Windows / Linux |
| --------------- | ------- | --------------- |
| New note        | `Cmd+N` | `Ctrl+N`        |
| Search          | `Cmd+K` | `Ctrl+K`        |
| Open graph view | `Cmd+G` | `Ctrl+G`        |
| Toggle sidebar  | `Cmd+\` | `Ctrl+\`        |
| Settings        | `Cmd+,` | `Ctrl+,`        |

## Editor

| Action          | macOS         | Windows / Linux |
| --------------- | ------------- | --------------- |
| Bold            | `Cmd+B`       | `Ctrl+B`        |
| Italic          | `Cmd+I`       | `Ctrl+I`        |
| Strikethrough   | `Cmd+Shift+S` | `Ctrl+Shift+S`  |
| Code            | `Cmd+E`       | `Ctrl+E`        |
| Insert link     | `Cmd+K`       | `Ctrl+K`        |
| Insert wikilink | `[[`          | `[[`            |
| Heading 1–3     | `Cmd+1/2/3`   | `Ctrl+1/2/3`    |
| Undo            | `Cmd+Z`       | `Ctrl+Z`        |
| Redo            | `Cmd+Shift+Z` | `Ctrl+Y`        |
| Save (manual)   | `Cmd+S`       | `Ctrl+S`        |

## Navigation

| Action       | macOS    | Windows / Linux |
| ------------ | -------- | --------------- |
| Back         | `Cmd+[`  | `Alt+Left`      |
| Forward      | `Cmd+]`  | `Alt+Right`     |
| Focus editor | `Escape` | `Escape`        |
