# What is Notesaner?

Notesaner is an open-source, web-first collaborative note-taking platform. It combines the local-first philosophy and powerful linking of Obsidian with the real-time collaboration and accessibility of Notion — while giving you full control over your data.

## Key Features

| Feature                     | Description                                                             |
| --------------------------- | ----------------------------------------------------------------------- |
| **Markdown-first editor**   | Write in Markdown with a rich WYSIWYG overlay powered by TipTap         |
| **Real-time collaboration** | Multiple users can edit the same note simultaneously via Yjs CRDT       |
| **Knowledge graph**         | Visualize connections between your notes with an interactive graph      |
| **Plugin system**           | Extend functionality with community plugins via a GitHub-based registry |
| **Self-hostable**           | Deploy on your own server with Docker Compose — your data, your rules   |
| **Open-source**             | MIT-licensed, transparent development, community-driven                 |

## How It Differs from Obsidian

- **Web-first**: No app install required — works entirely in the browser
- **Real-time collaboration**: Obsidian is local-first and single-user by default
- **Self-hosted server**: Notes stored on your server, not just local files

## How It Differs from Notion

- **Markdown files**: Notes are `.md` files on your filesystem — portable and durable
- **Open-source**: No vendor lock-in
- **Richer editing**: Full Markdown support with code blocks, front matter, and Mermaid diagrams
