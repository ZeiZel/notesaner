## Web App

Notesaner runs entirely in your browser — no installation required.

1. Navigate to your Notesaner instance URL (or [notesaner.com](https://notesaner.com) for the hosted version)
2. Click **Sign Up** and create an account
3. Create your first workspace

## Desktop App

A desktop app (Electron) is planned for a future release. Until then, the web app can be installed as a Progressive Web App (PWA) from your browser.

## Mobile App

See the [Mobile App](/help/getting-started/mobile) guide for iOS and Android instructions.

## First Workspace

After signing in, you'll be prompted to create a workspace. A workspace is the top-level container for all your notes and settings.

- Give your workspace a name
- Choose a storage location (for self-hosted instances)
- Invite collaborators (optional)
