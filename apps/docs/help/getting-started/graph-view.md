## Opening the Graph

Click the **Graph View** icon in the left sidebar, or press `Cmd+G` / `Ctrl+G`.

## Navigating the Graph

- **Click** a node to open the note
- **Drag** to pan the view
- **Scroll** to zoom in/out
- **Hover** over a node to see its connections highlighted

## Filtering

Use the filter panel to show only notes that match specific criteria:

- Tags
- Folders
- Date range
- Number of connections
