## Folders

Notesaner uses a folder tree in the left sidebar to organize notes. You can:

- Create folders and subfolders
- Drag and drop notes between folders
- Rename and delete folders

## Tags

Tags provide a flexible, cross-folder way to categorize notes. Add tags using `#tagname` anywhere in your note, or in the front matter:

```yaml
---
tags: [project, meeting, Q1-2026]
---
```

## Filtering by Tags

Click a tag in the sidebar tag panel to see all notes with that tag.
