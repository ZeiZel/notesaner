## Wikilinks

Type `[[` in the editor to trigger the note-linking autocomplete. Start typing the note title and select from the suggestions.

```markdown
See also: [[Meeting Notes 2026-03-29]]
```

## Link Aliases

Add a display label using the pipe syntax:

```markdown
[[Meeting Notes 2026-03-29|March Meeting]]
```

## Backlinks

Every note shows its **backlinks** — other notes that link to it — in the right sidebar panel.
