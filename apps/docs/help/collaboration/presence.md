# Presence Indicators

When multiple users have the same note open, their presence is shown via:

- **Avatar stack**: User avatars in the note toolbar header
- **Live cursors**: Colored cursor indicators showing where each user is typing
- **Selection highlights**: Other users' text selections are shown in their color

## Turning Off Live Cursors

If you find live cursors distracting, go to **Settings → Collaboration → Show live cursors** and toggle it off. Your cursor will still be visible to others.

## Presence in the Sidebar

The file tree shows small avatar indicators on notes that other users currently have open.
