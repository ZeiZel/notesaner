## Leaving a Comment

Select any text in a note and click the comment icon in the bubble menu (or press `Cmd+Shift+M` / `Ctrl+Shift+M`). Type your comment and press **Enter** to submit.

Comments appear as highlighted text with a thread indicator in the margin.

## Mentions

Type `@username` in a comment or in a note to mention a collaborator. They receive a notification.

## Resolving Comments

Once a discussion is resolved, click **Resolve** to collapse the thread. Resolved comments are hidden but can be viewed from the **Resolved** tab in the comments panel.
