This page is under construction. Complete documentation will be available soon.

A shared workspace is a Notesaner workspace with multiple members. All members see the same folder structure and notes (subject to permissions).

## Creating a Shared Workspace

1. Create a new workspace from the workspace switcher
2. Give it a name (e.g. "Engineering Team")
3. Invite your team members

## Workspace vs Personal Notes

- **Shared workspace**: All members can access all notes (per their role)
- **Personal notes**: Each user has a private workspace only they can access
